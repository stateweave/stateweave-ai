## Chosen mark: The Living Thread

The mark is one continuous path shaped into an abstract `S`. It is not a graph diagram, atom, sparkle, or brain.

- The open ends represent knowledge that can continue in either direction.
- The green thread represents accumulated context.
- The coral passage marks the active state moving through that context.
- The two soft reversals imply weaving while preserving a strong silhouette at 16px.
- The symbol works without the wordmark and remains recognizable in one color.

The logo intentionally avoids literal nodes. StateWeave's value is not that data is connected. Its value is that continuity stays useful while the active state changes.

## Usage

- Keep the green thread dominant.
- Coral may occupy only the active passage, never the full symbol.
- Do not add nodes, orbit rings, gradients, shadows, or an enclosing circle.
- Minimum digital size: 16px.
- Use the icon artwork in `app/icon.svg` and the interface component in `app/brand-mark.tsx`.
